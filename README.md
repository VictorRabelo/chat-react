# Chat React Frontend

Modern React frontend for a chat application. It consumes the [Chat API](https://github.com/VictorRabelo/chat-api-dotnet.git).

## Tech Stack

- React 18
- React Router
- Axios
- TailwindCSS
- Framer Motion
- Vite

## Getting Started

```bash
npm install
npm run dev
```

The app expects the API to run on `http://localhost:5000/api`.

## Project Structure

```
src/
  components/
  context/
  hooks/
  pages/
  services/
  utils/
```
